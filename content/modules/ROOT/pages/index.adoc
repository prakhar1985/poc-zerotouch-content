= LiteMaaS API Key Verification

This page verifies that LiteMaaS virtual key provisioning is working.

== Your LiteMaaS Credentials

[cols="1,3"]
|===
| *API URL* | `{litellm_api_url}`
| *API Key* | `{litellm_api_key}`
|===

== Quick Test

Open the terminal and run:

[source,bash,subs="attributes"]
----
curl -s {litellm_api_url}/v1/models \
  -H "Authorization: Bearer {litellm_api_key}" | python3 -m json.tool
----

You should see a list of available models.

== Chat Completion Test

[source,bash,subs="attributes"]
----
curl -s {litellm_api_url}/v1/chat/completions \
  -H "Authorization: Bearer {litellm_api_key}" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "granite-3-2-8b-instruct",
    "messages": [{"role": "user", "content": "Say hello in one sentence."}]
  }' | python3 -m json.tool
----
